# DatatablesSupportable

Add Datatables support (AJAX mode) to ActiveRecord

## Installation

Add this line to your application's Gemfile:

    gem 'datatables_supportable', :git=>"https://github.com/kkzhang/datatables_supportable"

And then execute:

    $ bundle
